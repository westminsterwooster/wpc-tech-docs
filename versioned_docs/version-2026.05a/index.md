# Insider Technology Documentation

This documentation is the internal reference for using and maintaining Westminster Presbyterian Church technology in Mackey Hall. It includes the current building-facing control instructions from the guest documentation plus the deeper operator references for livestreaming, cameras, computer workflows, and maintenance.

## Quick Start

1. Turn on the [sound system](documentation/audio/turning_sound_system_on.md) if microphones, computer audio, or Bluetooth audio are needed.
2. Lower the [projector screen](documentation/video/lowering_screen.md) if the projector will be used.
3. Turn on the [projector or rear TV](documentation/video/turning_projector_on.md) from the wall touchscreen.
4. Connect the source device and select the matching [video input](documentation/video/inputs.md).
5. Use the iPad [audio controls](documentation/audio/wall_audio_controls.md) to choose the right preset and unmute the channels needed.
6. Test picture and sound before the event or service begins.
